# ❓Quizz Trivial

A simple trivia game project with questions.

![HTML](https://img.shields.io/badge/HTML5-E34F26?logo=html5&logoColor=white)

## 🚀 Features

- Minimalist interface
- A series of 10 questions leading up to the results screen
- Questions that include both correct and incorrect options

## 📚 What I learned

- Manipulating the DOM
- Creating functions for various actions, such as selecting a question
